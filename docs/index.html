TG-Forward

强大的 Telegram 消息转发工具，支持多种通知渠道

查看文档 GitHub

功能特点

多渠道支持

支持钉钉、飞书、Bark、HarmonyOS 等多个通知渠道，轻松转发消息到不同平台

消息持久化

支持内存队列和 LevelDB 持久化存储，确保消息不丢失，支持断点续传

灵活配置

提供丰富的配置选项，支持自定义通知规则、消息格式和过滤条件

监控指标

集成 Prometheus 监控，提供消息处理延迟、队列长度等关键指标的实时监控

安全可靠

支持消息签名验证、自定义加密、访问控制，确保消息传输安全

插件扩展

提供插件系统，支持自定义消息处理逻辑，轻松扩展新的通知渠道

支持的通知渠道

钉钉机器人

支持文本、Markdown、图片等多种消息类型

飞书机器人

支持富文本、图片、@功能等高级特性

Bark

iOS 设备推送通知，支持自定义声音和图标

HarmonyOS

华为设备消息推送，支持多用户配置

使用文档

快速开始

1. 克隆仓库：

git clone https://github.com/user/tg-forward-to-xx.git

2. 配置文件：

cp config/config.example.yaml config/config.yaml
vim config/config.yaml

3. 编译运行：

go build -o tg-forward
./tg-forward -config config/config.yaml

配置说明

支持以下配置项：

  • Telegram Bot Token 和群组 ID
  • 钉钉机器人 Webhook 和密钥
  • 飞书机器人 Webhook 和密钥
  • Bark 设备 Token
  • HarmonyOS 推送配置
  • 消息队列类型（内存/LevelDB）
  • 日志级别和输出位置
